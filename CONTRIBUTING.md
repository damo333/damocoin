# Contributing to DamoCoin

Thanks for your interest in contributing to DamoCoin! This project is a simple Solana SPL token sample, and contributions are welcome in the form of bug fixes, documentation improvements, or new examples.

## How to contribute

1. Fork the repository.
2. Create a new branch for your change:
   ```bash
git checkout -b my-feature-name
```
3. Make your changes and test them locally.
4. Commit your changes with a clear message.
5. Push the branch to your fork and open a pull request.

## What to contribute

- README improvements or clearer instructions
- better sample metadata or asset hosting examples
- additional script features for DamoCoin
- GitHub Pages or IPFS automation helpers

## Code style

- Keep scripts simple and readable
- Use consistent indentation and comments
- Keep examples compatible with Node.js and Solana devnet

## Reporting issues

If you find a bug or have a feature request, please open an issue with as much detail as possible, including:
- what you were trying to do
- the exact command you ran
- the output or error message
- the file and line if relevant
