FAQ
What is the Damo OG Collection?
The Damo OG Collection is a set of 100 NFTs on Solana, available exclusively to DAMO holders. Each NFT costs 10 DAMO to mint via the mint page. The collection is powered by Metaplex Candy Machine v3 with an on-chain token payment guard.
What is DamoCoin?
DamoCoin (DAMO) is a custom SPL token deployed on Solana Mainnet. It was built using the Solana Web3.js library and the SPL Token program, with on-chain metadata registered via the Metaplex standard. The complete source code is available on GitHub.
Where can I buy DAMO?
DAMO is available on Raydium. Connect your Phantom or Solflare wallet and swap SOL for DAMO. See the How to Buy section above for step-by-step instructions.
What is the total supply?
The total supply is approximately 600 DAMO. The original supply of 1,000 was reduced by burning 400 DAMO on-chain. The supply is permanently fixed — mint authority has been revoked. Full tokenomics are available on the Tokenomics page.
Was there a presale or ICO?
No. There was no presale, private sale, or ICO. All tokens were minted to the creator wallet at launch. A portion was deposited into the Raydium liquidity pool to enable trading.
Which wallets support DAMO?
Any Solana-compatible wallet supports DAMO, including Phantom and Solflare. Thanks to on-chain Metaplex metadata, the token appears with its name and logo automatically.
Is the project open source?
Yes. The complete source code including token creation scripts, metadata setup, and this website is publicly available on GitHub.
